The portfolio page you are currently browsing is also a project piece! It is a fully custom-built site put together to showcase our work, projects and experiences. Built using Astro, React and Typescript, the site is flexible, easily extensible and serves our needs perfectly. Take a look yourself!

A custom-built portfolio to showcase our projects, experience, and skills.
Designed to be simple, clear, and easily expandable, this site serves as both a personal introduction and a technical demonstration of my development abilities.

This portfolio isn’t just a collection of projects—it’s a project itself!
I built it to experiment with modern web development, structure content efficiently, and provide a clean, accessible way to present my work (and also a bit out of necessity).

Features

  • 🔹 Dynamic Content Handling—Project descriptions and experience are stored in Markdown and dynamically loaded.
  • 🎨 Minimalist & Responsive Design—A clean layout that works (hopefully) on both desktop and mobile.
  • ⚡ Lightweight—Optimized for readability.
  • 🌍 Custom Domain & Deployment—The site is hosted using GitHub Pages and linked to a personal domain.

To succeed, both players must communicate effectively under pressure. One describes the problem, the other searches for the solution. Simple? Think again.

Features

  • 🚀 Immersive spaceship cockpit—interact with buttons, switches, dials, and complex systems.
  • 📖 Physical manual gameplay—one player relies entirely on a paper guide to help the pilot navigate chaos.
  • ⏳ Fast-paced decision-making—react to emergencies before the ship falls apart.
  • 🎙️ Hilarious and chaotic communication—explain, interpret, and (hopefully) not misunderstand each other.
  • 🌌 Varied space environments—asteroid fields, nebulae, enemy encounters, and more.

Technology Used

🛠️ Built with:

  • Astro for fast development and static page builds.
  • TypeScript & React for a modular and scalable frontend.
  • Markdown Parsing to manage content dynamically.
  • Tailwind & Daisy-UI for a flexible, modern UI.
  • GitHub Pages for hosting.

What I learned

This project allowed us both to, again, review current web development frameworks and dust off our long forgotten web development skills - Typescript, React, web deployment, …

  • Working with Astro - A nice surprise, that writing webs does not have to be ugly…
  • Long forgotten knowledge of React.
  • Styling using Tailwind and Daisy-UI, where you don’t have to do everything from scratch :)

This website was built to professionally showcase our own projects and work experience.
